Factor Analysis: Identifying Latent Variables

Loading analysis...
Objective

This factor analysis project explores the underlying dimension structure in multivariate data with 9 variables. Using principal component analysis and factor rotation techniques, I identified 3 key latent factors that explain 62.4% of the total variance in the dataset.

Tools & Techniques
Python Principal Component Method None Rotation Eigenvalue Analysis
Key Findings
  • 1. Identified 3 significant factors explaining 62.4% of total variance in the dataset
  • 2. Kaiser-Meyer-Olkin (KMO) value of 0.703 confirmed good sampling adequacy
  • 3. Bartlett's test showed significant correlations (p ≈ 5.99e-153)
  • 4. Extracted Factor 1 (Quality of Life/Amenities), Factor 2 (Socioeconomic), and Factor 3 (Climate)
