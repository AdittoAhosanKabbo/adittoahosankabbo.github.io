Objective

This factor analysis project explores the underlying dimension structure in multivariate data with 9 variables. Using principal component analysis and factor rotation techniques, I identified 3 key latent factors that explain 62.4% of the total variance in the dataset.

Tools & Techniques
Python Principal Component Method None Rotation Eigenvalue Analysis
Key Findings
  • Identified 3 significant factors explaining 62.4% of total variance in the dataset
  • Kaiser-Meyer-Olkin (KMO) value of 0.703 confirmed good sampling adequacy
  • Bartlett's test showed significant correlations (p ≈ 5.99e-153)
  • Extracted factors (Without Rotation):
    • Factor 1: Quality of Life/Amenities
    • Factor 2: Socioeconomic
    • Factor 3: Climate
