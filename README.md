# Aditto Ahosan Kabbo - Portfolio Website

A professional portfolio website showcasing my skills, projects, and experience as a Data Analyst and Statistics Student.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Sections for showcasing skills, education, projects, and contact information
- Interactive contact form
- Social media integration

## Technologies Used

- HTML5
- CSS3 (with custom variables and animations)
- JavaScript (vanilla JS with no frameworks)
- Font Awesome for icons
- Google Fonts

## Project Structure

```
portfolio-website/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styling
├── js/
│   └── script.js       # JavaScript functionality
└── images/             # Project images and other media
```

## Setup and Deployment

1. Clone or download this repository
2. Open `index.html` in your web browser to view locally
3. Upload to any web hosting service to deploy online
